# Project Roadmap

This roadmap outlines the planned development phases for Portfolio v2, the goal is to continuously evolve the platform while experimenting with modern web technologies and engineering practices.

---

## Phase 1 — Foundation

Goals:

- Monorepo setup
- Portfolio application initialization
- Documentation platform
- Initial deployment

Features:

- Next.js portfolio app
- Docusaurus documentation system
- GitHub repository integration
- Basic UI layout

---

## Phase 2 — Portfolio Core Features

Goals:

- Showcase projects dynamically
- Improve portfolio presentation

Features:

- GitHub API integration
- Dynamic project cards
- Skills and experience sections
- Project filtering

---

## Phase 3 — Engineering Content

Goals:

- Document engineering practices
- Showcase architecture decisions

Features:

- Architecture documentation
- Case studies for projects
- Technical blog posts

---

## Phase 4 — Optimization

Goals:

- Improve performance and SEO

Features:

- SEO optimization
- metadata and OpenGraph
- performance improvements

---

## Future Ideas

Possible future additions:

- analytics
- github contributions graph
- building automation stages 
- project case studies
- interactive architecture diagrams
- experimental projects
