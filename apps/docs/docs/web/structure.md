# Portfolio v2 - Web Application Structure

## Directory Layout

```
apps/web
│
├── app
├── components
├── lib
├── public
└── styles
```

### app/

Contains page routes and layouts.

Example:

```
- app/page.tsx
- app/projects/page.tsx
```

### components/

Reusable UI components.

Examples:

- navigation

- project cards

- hero section

### lib/

Utility functions and integrations.

Examples:

- GitHub API functions

- Helper utilities
